# 🛍️ Drip Store

Loja virtual moderna desenvolvida com foco em experiência do usuário, performance e design responsivo.

Este projeto simula um e-commerce de roupas estilo streetwear, com interface limpa e navegação intuitiva.

---

## 🚀 Tecnologias Utilizadas

* ⚛️ React
* ⚡ Vite
* 🎨 CSS3
* 🧠 JavaScript (ES6+)

---


---

## 💻 Funcionalidades

* 🛒 Listagem de produtos
* 🔍 Interface intuitiva
* 📱 Design responsivo (mobile e desktop)
* ⚡ Carregamento rápido com Vite
* 🎯 Estrutura organizada para escalabilidade

---

## 📦 Como rodar o projeto

```bash
# Clone o repositório
git clone https://github.com/MaiaraMendes25/Loja-drip.git

# Acesse a pasta do projeto
cd Loja-drip

# Instale as dependências
npm install

# Execute o projeto
npm run dev
```
